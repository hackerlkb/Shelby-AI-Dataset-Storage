# Shelby Storage Demo

A simple Web3 demo application that shows how Shelby can be used as a decentralized storage layer.

This project demonstrates how files and datasets can be stored and managed using decentralized infrastructure.

## Features

- Upload files to decentralized storage
- Retrieve stored files
- Simple Node.js demo

## Tech Stack

- Node.js
- Web3
- Shelby decentralized storage

## Use Cases

Shelby can power many decentralized applications such as:

- AI dataset storage
- decentralized file sharing
- Web3 application data storage

## Installation

```bash
git clone https://github.com/yourname/shelby-storage-demo.git
cd shelby-storage-demo
npm install
```

## Run

```bash
node index.js
```
